# NFeCloud - SDK PHP


## Descrição
Este pacote consiste em um SDK em PHP para a utilizacao do sistema NFeCLoud através de api - REST.

# Requisitos
- PHP >=5.5.19;


# Instalação

Via Composer

```bash
composer require SbaumTecnologia/nfecloud-php
```
